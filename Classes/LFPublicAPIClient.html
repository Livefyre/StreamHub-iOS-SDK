Parameters

userId

The Id of the user whose content is to be fetched.

(optional) The lftoken of the user whose content is to be fetched. This parameter is required by default unless the network specifies otherwise.

The network to query agianst as identified by domain, i.e. livefyre.com.

offset

(optional) Number of results to skip, defaults to 0. 25 items are returned at a time.

success

Callback called with a dictionary after the results data has been retrieved.

failure

Callback called with an error after a failure to retrieve data.

(optional) CSV of comment states to return.

void
