getContentForPage:withInitInfo:onSuccess:onFailure:

The init data for a collection contains information about that collection’s number of pages and where to fetch the content for those pages.

+ (void)getContentForPage:(NSUInteger)pageIndex withInitInfo:(NSDictionary *)initInfo onSuccess:(void ( ^ ) ( NSDictionary *contentInfo ))success onFailure:(void ( ^ ) ( NSError *error ))failure

Parameters

pageIndex

The page to fetch content for. The pages are numbered from zero. If the page index provided is outside the bounds of what the init data knows about the error callback will convey that failure.

initInfo

A pointer to a bootstrap’s init data as returned by the bootstrap init endpoint and parsed from JSON into a dictionary.

success

Callback called with a dictionary represting the pages content.

failure

Callback called with an error after a failure to retrieve data.

Return Value

void
